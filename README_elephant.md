# README Elephant

## Install

```bash
pnpm i --no-frozen-lockfile
```

## Build

```bash
pnpm build
```

## Execution

Start the server with the elephant character by running:

```bash
pnpm start --character="characters/elephant.character.json"
```
in other terminal, start the client:

```bash
pnpm start:client
```

Open the client and test
